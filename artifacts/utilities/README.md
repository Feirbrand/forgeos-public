## Quick Start

### Prerequisites
```bash
# Install dependencies
pip install -r ../requirements.txt

# Required: NumPy 1.26+, Pandas 2.0+, Matplotlib 3.8+
```

### Basic Operations

**SIF Detection (Symbolic Identity Fracturing)**:
```bash
# Real-time monitoring
python sif-check.py --data sample_logs.csv --threshold 0.7

# Enterprise deployment with alerting
python sif-check.py --continuous --alert-level high
```

**Torque Simulation (Cascade Modeling)**:
```bash
# Basic cascade simulation
python torque_sim_clean.py --iterations 1000 --threshold 0.7

# Advanced analysis with metrics export
python torque_sim_clean.py --analysis-mode --export results.csv
```

**OBMI Harmony Sync (Memory Validation)**:
```bash
# Twin memory harmony analysis
python obmi-harmony-stub.py --mode validation

# Production harmony monitoring
python obmi-harmony-stub.py --continuous --threshold 0.8
```

**Strategic Role Analysis**:
```bash
# Automated role analysis
python sra-playbook-stub.py --full-analysis

# Drift purge automation
python srd_purge_stub.py --purge-mode --safety-checks
```
