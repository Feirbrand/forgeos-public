## Quick Start

### Prerequisites
```bash
# Install dependencies
pip install -r requirements.txt

# For Jupyter notebooks
pip install jupyter lab

# Required: NumPy 1.26+, Matplotlib 3.8+, Pandas 2.0+
```

### Interactive Analysis

**Behavioral Pattern Recognition**:
```bash
# Launch Jupyter Lab
jupyter lab notebooks/behavioral-pattern-recognition.ipynb

# Run GWI calculations
python notebooks/gwi_calc_clean.py --mode interactive
```

**Torque Healing Simulation**:
```bash
# Interactive simulation
jupyter lab notebooks/torque-healing-sim.ipynb

# Cross-platform validation
jupyter lab notebooks/cross-platform-validation.ipynb
```

**VX Threat Simulation**:
```bash
# VX Ghost Echo analysis
jupyter lab notebooks/vx-ghost-echo-sim.ipynb

# Drift detection monitoring
jupyter lab notebooks/drift-detection.ipynb
```

### Threat Intelligence

**Perfect Mimic Analysis**:
```bash
# Review threat documentation
cat threat-intelligence/perfect-mimic-family/vx-ghost-echo.md

# CTTA framework examples
cat threat-intelligence/mimics_ctta_example.md
```

### Diagnostic Utilities

**Real-Time Monitoring**:
```bash
# SIF detection
python utilities/sif-check.py --threshold 0.7 --continuous

# Torque stability analysis
python utilities/torque_sim_clean.py --iterations 1000
```

**RAY Framework Validation**:
```bash
# Validate RAY v2.1 installation
python utilities/ray-validator.py --test-all

# DNA Codex integration test
python utilities/ray-codex-loader.py --verify-strains --codex-version 5.4
```

**CSFC Framework Integration**:
```bash
# Complete methodology review
cat utilities/csfc_chain_overview.md

# OBMI harmony validation
python utilities/obmi-harmony-stub.py --mode analysis
```
