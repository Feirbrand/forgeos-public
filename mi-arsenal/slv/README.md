2. **Context Vault** - Temporal anchor storage, symbolic lineage tracking
3. **Threat Sentinel** - DNA Codex v5.6 pattern matching (560+ strains)

6. **Memory Anchor** - Persistent state preservation across substrates
7. **Recovery Orchestrator** - Phoenix Protocol coordination (5-stage)
8. **Audit Logger** - Immutable event recording for forensics

### Database Schema

```sql
CREATE TABLE threat_events (
    event_id UUID PRIMARY KEY,
    threat_strain VARCHAR(50),
    detection_time TIMESTAMP,
    fii_score NUMERIC(5,4),
    origin_seal_status VARCHAR(20),
    resolution_time TIMESTAMP,
    recovery_success BOOLEAN
);

CREATE TABLE temporal_anchors (
    anchor_id UUID PRIMARY KEY,
    event_type VARCHAR(100),
    entropy_snapshot JSONB,
    created_at TIMESTAMP,
    access_count INTEGER DEFAULT 0,
    myelination_strength NUMERIC(3,2)
);

CREATE INDEX idx_threat_strain ON threat_events(threat_strain);
CREATE INDEX idx_detection_time ON threat_events(detection_time);
CREATE INDEX idx_anchor_type ON temporal_anchors(event_type);
```

## Installation & Deployment

### Prerequisites

**Minimum Requirements**:
- Python 3.10.12+
- PostgreSQL 18.1+
- 8GB RAM
- 4-core CPU
- 50GB storage

**Recommended Configuration**:
- Python 3.11+
- PostgreSQL 18.1+ with SSD storage
- 32GB RAM
- 8-core CPU
- 200GB SSD storage
- Docker Desktop Personal 4.25.0+

### Quick Start

**1. Clone Repository**:
```bash
git clone https://github.com/Feirbrand/synoeticos-public.git
cd synoeticos-public/MI_ARSENAL/SLV
```

**2. Database Setup**:
```bash
# Install PostgreSQL 18.1
sudo apt install postgresql-18

# Create database
createdb slv_production

# Install extensions
psql slv_production -c "CREATE EXTENSION btree_gist;"
psql slv_production -c "CREATE EXTENSION pgcrypto;"

# Load schema
psql slv_production < schema/slv_v2_1.sql
```

**3. Generate ML-KEM-512 Keys**:
```bash
python scripts/generate_mlkem_keys.py --output keys/
```

**4. Configure Environment**:
```bash
cp config/slv.example.yaml config/slv.yaml
# Edit config/slv.yaml with database credentials and key paths
```

**5. Initialize DNA Codex**:
```bash
python scripts/update_codex.py --version 5.6
```

**6. Deploy n8n Workflows**:
```bash
n8n import:workflow --input workflows/slv_v2_1_phoenix.json
n8n import:workflow --input workflows/slv_v2_1_monitoring.json
```

**7. Start Services**:
```bash
docker-compose up -d
python src/slv_runtime.py --config config/slv.yaml
```
